# Nutri-Project
Food Classification Using Nutritional Data (Machine Learning Project)

## Project Overview
This project classifies food items based on nutritional attributes such as calories, protein, fat, carbohydrates, and sugar using machine learning models.

## Techniques Used
- Data Preprocessing
- Feature Engineering
- Multi-class Classification
- Model Evaluation

## Models Implemented
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest achieved the highest accuracy and was selected as the final model.

## Tools & Libraries
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
